3) Flujo recomendado

1. Registrar movimiento(s) en data/movimientos.csv. 2. Consultar consecutivo: python src/consecutivo.py 3. Generar HTML: python src/generar_html.py 4. Convertir a PDF: python src/generar_pdf.py 5. Revisar resumen: python src/resumen.py

Tip: abre este archivo en tu navegador y úsalo como panel operativo diario. No reemplaza la CLI; la complementa.
